Flexbox - One-Dimensional Layout

Basic Flex Container

/* FLEXBOX BASICS */
.container {
    display: flex;           /* or inline-flex */
    flex-direction: row;     /* row | row-reverse | column | column-reverse */
    flex-wrap: wrap;         /* nowrap | wrap | wrap-reverse */
    justify-content: center; /* Main axis alignment */
    align-items: center;     /* Cross axis alignment */
    gap: 1rem;               /* Space between items */
}

Flex Item Properties

.item {
    flex-grow: 1;    /* How much item should grow (default: 0) */
    flex-shrink: 1;  /* How much item should shrink (default: 1) */
    flex-basis: auto; /* Initial size before growing/shrinking */
    
    /* Shorthand */
    flex: 1;         /* flex-grow: 1, flex-shrink: 1, flex-basis: 0% */
    flex: 1 1 200px; /* grow shrink basis */
    
    align-self: center; /* Override align-items for this item */
    order: 2;           /* Change visual order */
}

Common Flexbox Patterns

/* Center anything */
.center-everything {
    display: flex;
    justify-content: center;
    align-items: center;
}

/* Sticky footer */
.page {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}
.main { flex: 1; }

/* Equal height columns */
.columns {
    display: flex;
}
.column { flex: 1; }

/* Navigation with logo left, links right */
.nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

CSS Grid - Two-Dimensional Layout

Basic Grid Container

/* GRID BASICS */
.container {
    display: grid;
    grid-template-columns: 200px 1fr 200px;  /* 3 columns */
    grid-template-rows: auto 1fr auto;        /* 3 rows */
    gap: 1rem;                                /* Row and column gap */
    /* or */
    row-gap: 1rem;
    column-gap: 2rem;
}

Responsive Grid with auto-fit/auto-fill

/* Auto-fit: Columns stretch to fill space */
.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 1rem;
}

/* Auto-fill: Creates empty columns if space available */
.grid {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    gap: 1rem;
}

Grid Item Placement

.item {
    /* Line-based placement */
    grid-column: 1 / 3;      /* Start at line 1, end at line 3 */
    grid-row: 1 / 2;
    
    /* Span syntax */
    grid-column: span 2;     /* Span 2 columns */
    grid-row: span 3;        /* Span 3 rows */
    
    /* Shorthand */
    grid-area: 1 / 1 / 3 / 3; /* row-start / col-start / row-end / col-end */
}

Flexbox vs Grid - When to Use

/**
 * USE FLEXBOX FOR:
 * - Navigation bars
 * - Card layouts (single row/column)
 * - Centering content
 * - Distributing space in one dimension
 * - When content size should determine layout
 * 
 * USE GRID FOR:
 * - Page layouts
 * - Complex two-dimensional layouts
 * - When you need precise control over rows AND columns
 * - Overlapping elements
 * - When layout should determine content size
 * 
 * COMBINE THEM:
 * - Grid for overall page layout
 * - Flexbox for components within grid areas
 */

üé§ Interview Questions

  • Flexbox vs Grid? Flexbox is 1D (row OR column), Grid is 2D (rows AND columns)
  • What does flex: 1 mean? flex-grow: 1, flex-shrink: 1, flex-basis: 0%
  • auto-fit vs auto-fill? auto-fit stretches items, auto-fill creates empty tracks
  • How to center with Flexbox? justify-content: center; align-items: center;
  • What is fr unit? Fraction of available space in grid container
